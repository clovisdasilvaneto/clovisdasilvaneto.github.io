Bla bla bla... Vai logo pra parte em que você fala como foi feito o blog

O blog foi desnvolvido usando um gerador estático, por preferência usei o Metalsmith, pela gama de plugins e fácilidade no desenvolvimento que ele me oferece. Também pensei em usar o Jkylle, mas optei pelo Metalsmith, depois de ter vistos alguns post's fantásticos do meu amigo Palmer Oliveira.

As demais tecnologias utlizadas foram:

  • Gulp Js - Para mimificar o js, compilar o SASS e mimificar o css.

  • Sass - Bem, já fazia tmepo que eu estava prometendo o blog, então usei o Sass para poder agilizar o processo de desenvolvimento e também pelos mixins que eu particulamente sou hiper-mega-fã.

  • Github Pages - Onde está hospedado o blog :)

  • Node Js - "O queeeê ?! mas você não falou agora pouco que é um blog estático, e como você conseguiu hospedar um blog rodando no NodeJs no github O.o ????". Caalma pessoal, o blog é estático sim, e não tem nada de NodeJs no github kkkk, mas o Metalsmith é javascript e roda com NodeJs, mas tudo isso na minha máquinha, quando eu quero passar pra o github é só executar um node index na raiz do meu diretório, que ele gera os arquivos html pra mim, tudo simples e prático.

  • Handlebars - Template engine do blog.

Também estou utilizando o metalsmith-markdown, que é um plugin do metalsmith interpreta os meus arquivos .md
